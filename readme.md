# soli

My implementation of ls

## Installation

Clone the repo:

```sh
git clone https://github.com/JeffTheK/soli
```

Then run `make install` in the project directory

```sh
make install
```

## Usage

Just run soli in terminal

## Building

```sh
make install
```

### Tests

```sh
make test
```
